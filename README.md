# Alianza Para el Progreso (AppLaravel)

Repositorio oficial del portal institucional de Alianza Para el Progreso, desarrollado con Laravel 11 y React.

## Tecnologías
- **Backend**: Laravel 11 (PHP 8.2+)
- **Frontend**: React + Inertia.js
- **Estilos**: Tailwind CSS v4
- **Iconos**: Lucide React
- **Estado**: Desarrollado con Antigravity

## Instalación
1. Clonar el repositorio.
2. `composer install`
3. `npm install`
4. Configurar el archivo `.env` (usar `.env.example`).
5. `php artisan key:generate`
6. `npm run build` o `npm run dev`

## Despliegue
Este proyecto está optimizado para su despliegue en entornos de producción.
